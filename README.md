# PromptCraft AI

AI Prompt Engineering and Optimization Platform designed to help users create, test, organize, and improve prompts for various AI-powered applications.

## Features

- Prompt creation and management
- Prompt optimization suggestions
- Reusable prompt templates
- Prompt evaluation and scoring
- AI API integration
- Export and documentation support

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- OpenAI API / Gemini API
